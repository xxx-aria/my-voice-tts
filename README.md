# Local Voice TTS MVP

Локальный учебный проект для озвучки текста голосом, похожим на мой собственный голос.

⚠️ Используй только свой голос или голос человека, который явно дал разрешение.

## Что используется

- Python 3.11
- Gradio
- Coqui TTS / XTTS-v2
- PyTorch
- Локальная генерация .wav
